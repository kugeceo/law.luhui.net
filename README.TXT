建议：

系统配置：
    P5/133以上CPU
    16M以上内存
    剩余硬盘空间50M以上
    4速以上光驱
    WINDOWS NT/95/98/2000/XP
    INTERNET EXPLORER 4.0或以上
    16位真彩色
    显示分辨率1024 X 768

使用说明：
    插入本光盘, 双击本光盘下INDEX.HTM。


本光盘分为四大部分：

    1、常用经济贸易示范合同：收录了各种经济贸易示范合同文本近100种。为便于使用和修改，在每份
合同尾部有“下载”链接，点击即可下载该合同文本的WORD格式。

    2、法律与国际公约：收录了常用的法律法规、国际公约和惯例。由于法律经常修订，最新的文本请
登录中国国际经济贸易仲裁委员会华南分会的网站查看。网址为：http://www.sccietac.org

    3、经济贸易案例分析：收录了部分中国国际经济贸易仲裁委员会华南分会裁决的案例。
    
    4、仲裁指南：包括了中国国际经济贸易仲裁委员会华南分会的介绍、为什么选择仲裁、为什么
选择CIETAC仲裁、如何订立仲裁条款或协议、如何办理仲裁申请、仲裁程序、如何执行仲裁裁决等内容。

    在使用的过程中，如有任何问题或建议，欢迎通过电子邮件向我们提出: info@sccietac.org
